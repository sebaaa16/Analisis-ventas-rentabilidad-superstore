# Analisis-ventas-rentabilidad-superstore
Dashboard interactivo en Power BI para el análisis de ventas y rentabilidad de un dataset de retail (E-commerce). El proyecto abarca desde la limpieza de datos (ajuste de escalas y tipos) hasta la creación de KPIs, análisis geográfico y tendencias temporales para identificar oportunidades de mejora en el margen operativo
